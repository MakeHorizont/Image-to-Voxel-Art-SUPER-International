# 图像转体素艺术 (Image to Voxel Art) 🌍🧊

[🔙 返回主 README](../README.md)

**利用 Google Gemini 3 的强大功能，将任何图像转换为完全交互式的 3D 体素艺术场景。**

此应用程序使用先进的 AI 分析 2D 图像，并“幻想”出深度、结构和材质，从而使用 Three.js 实时构建 3D 世界。它支持用于探索的“飞行模式”，并可导出为 GLB (Blender/Unity) 等专业格式。

---

## ✨ 功能
*   **图像转 3D：** 上传任何图像或使用文本描述来生成体素场景。
*   **交互式查看器：**
    *   **轨道模式：** 围绕对象旋转查看。
    *   **飞行模式：** 使用 WASD + 鼠标控制在场景中飞行（支持移动端触摸控制！）。
*   **导出功能：**
    *   **HTML：** 将场景保存为独立的网页文件。
    *   **GLB (Binary GLTF)：** 直接导出到 Blender、Unity、Godot 或 Unreal Engine。
*   **代码编辑：** AI 提供完整的 Three.js 代码，您可以对其进行编辑或复制。

## 🎮 控制
*   **轨道：** 左键点击 + 拖动以旋转。滚动以缩放。
*   **飞行模式：**
    *   `W` `A` `S` `D` 移动。
    *   `Q` `E` 上升/下降。
    *   `Shift` 开启加速模式。
    *   **移动端：** 勾选 "Touch" 复选框以使用虚拟操纵杆。
